###Ubuntu 下设置开始启动项


可以通过一些小工具来管理Ubuntu的启动项：

小工具 `rcconf`
安装rcconf

```bash
sudo apt-get install rcconf
```

运行rcconf

```bash
sudo rcconf
```

![ubuntu_rcconf](http://7xifyp.com1.z0.glb.clouddn.com/ubuntu_rcconf.png)

在选项上按空格，会取消或设置启动项。
